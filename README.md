# 🇩🇪 German Gambler

**German Gambler** is a modern web application designed to revolutionize the way you learn German. By combining AI-powered insights with gamified elements, it moves away from boring rote memorization and focuses on context, real-world usage, and interactive challenges.

---

## 🚀 Key Features

### 🤖 AI Word Explanations
Get smart, context-aware explanations for German vocabulary. The system provides deep insights into linguistic nuances and provides practical, real-world examples.

## 🛠️ Tech Stack

This project leverages a robust and modern development stack:

* **Backend:** [Django](https://www.djangoproject.com/) (Python)
* **Frontend:** [Tailwind CSS](https://tailwindcss.com/)
* **Template Engine:** Django Templates
* **Icons & UI:** FontAwesome & Custom SVG
* **Interactivity:** Vanilla JavaScript (Mobile burger menu, UI overlays)

---

## 📋 Project Structure

The application is organized into three primary modules:

1.  **Home:** Feature overview and the interactive "Gamble Word" engine.
2.  **Learn:** A dedicated space for active vocabulary acquisition and AI interaction.
3.  **Dictionary:** A searchable personal repository of all encountered words and phrases.
