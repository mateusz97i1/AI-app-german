# 🇩🇪 German Gambler

**German Gambler** is a modern web application designed to revolutionize the way you learn German. By combining AI-powered insights with gamified elements, it moves away from boring rote memorization and focuses on context, real-world usage, and interactive challenges.

---

## 🚀 Key Features

* **AI Word Explanations:** Get smart, context-aware explanations for German vocabulary, including nuances and real-world examples.
<img width="1858" height="850" alt="1" src="https://github.com/user-attachments/assets/e60e7955-61bc-4733-ac57-673510eb1f5e" />

* **Gamble Word:** A core interactive module where you "gamble" on new words, making the discovery of vocabulary unpredictable and engaging.
* **Mini Games:** Test your knowledge with quick, addictive "Guess the Word" challenges designed to reinforce memory.
<img width="890" height="818" alt="2" src="https://github.com/user-attachments/assets/4c45e6f0-3c49-4ffd-b143-8d0b2b27ed7b" />
* **Dynamic Dictionary:** Access an ever-growing, searchable database of words with live updates and usage contexts.
<img width="1778" height="829" alt="3" src="https://github.com/user-attachments/assets/7e5dbc8b-ad0b-4227-b4cb-c7f9b00bc580" />
* **Responsive Design:** A mobile-first UI built with Tailwind CSS, featuring smooth transitions and a sleek glassmorphism overlay.

---

## 🛠️ Tech Stack

This project is built using a robust and modern stack:

* **Backend:** [Django](https://www.djangoproject.com/) (Python)
* **Frontend:** [Tailwind CSS](https://tailwindcss.com/)
* **Template Engine:** Django Templates
* **Icons & UI:** FontAwesome & Custom SVG
* **Interactivity:** Vanilla JavaScript (Mobile burger menu, UI overlays)

---

## 📋 Project Structure

The application is divided into three main sections:
* **Home:** Overview of features and the "Gamble Word" interactive section.
* **Learn:** Dedicated space for active vocabulary acquisition.
* **Dictionary:** A searchable repository of all previously encountered words.
